# 🕵️ Cyber Scavenger Hunt

A beginner-friendly cybersecurity CTF (Capture The Flag) challenge playground — powered by Docker & Flask.

## 🚀 Features

- AI Prompt Injection
- SQL Injection
- Cross-Site Scripting (XSS)
- Broken Authentication
- Web-based Homepage UI
